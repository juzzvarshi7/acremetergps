🌾 AcreMeter GPS App

A high-performance Android GPS tracking application built during internship at Sun Industrial Automation & Solutions Pvt. Ltd. by Wolg.

📅 Internship Duration

4th April 2025 – 12th June 2025

🛠 Tech Stack

Java Kotlin DSL SQLite OSMDroid LocationManager Retrofit Flask MySQL Bluetooth

📱 App Highlights

  • Login & Dashboard — Kotlin DSL and SQLite for local authentication and UI routing.
  • GPS Tracking — Built in Java using LocationManager to record routes and store them as TXT.
  • Trip Replay — OSMDroid with animated polylines and a moving tractor icon.
  • Sync to Device — Uses BluetoothSocket to transfer trip logs to connected hardware.
  • Trip Data Viewer — Integrated with Flask API and MySQL using Retrofit for summary and logs.
  • Trip Actions — Users can preview maps, download trip data as CSV, and view metadata.
