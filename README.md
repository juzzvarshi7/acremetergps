AcreMeter is an Android-based land measurement and tracking application built with real-time GPS, map visualization, and data storage capabilities. It enables users to measure land accurately, track movement paths, and replay recorded routes directly on the map.

## 🚀 Core Features

📍 Real-time GPS tracking to capture land boundaries
🗺️ Map-based plotting (OSMDroid) for precise area marking
📏 Calculate area, distance, and perimeter dynamically
▶️ Trip replay feature with animated path visualization
💾 Local database storage (SQLite/Room) for saving measurements
🔄 Data sync & transfer (Bluetooth / server-ready architecture)
📊 Structured data handling (split data packets, metadata storage)

## ⚙️ Tech Highlights

Android (Java/Kotlin)
OSMDroid for map rendering
GPS + Polyline tracking & animation
Room Database for local persistence
Bluetooth communication for device integration

## 🎯 Practical Use Cases
Farm land measurement with live tracking
Route/path analysis and replay
Field data collection and storage
Smart surveying with offline capability
